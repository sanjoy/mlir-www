28 /// `MaybeLocalRepr` contains the indices of the contraints that can be
29 /// expressed as a floordiv of an affine function. If it's an `equality`
30 /// contraint `equalityIdx` is set, in case of `inequality` the `lowerBoundIdx`
31 /// and `upperBoundIdx` is set. By default the kind attribute is set to None.
32 struct MaybeLocalRepr {

35  unsigned equalityIdx;

37  unsigned lowerBoundIdx, upperBoundIdx;
38  } inEqualityPair;
39  } repr;
40 };

55 /// Given dividends of divisions `divs` and denominators `denoms`, detects and
56 /// removes duplicate divisions. `localOffset` is the offset in dividend of a
57 /// division from where local identifiers start.
58 ///
59 /// On every possible duplicate division found, `merge(i, j)`, where `i`, `j`
60 /// are current index of the duplicate divisions, is called and division at
61 /// index `j` is merged into division at index `i`. If `merge(i, j)` returns
62 /// `true`, the divisions are merged i.e. `j^th` division gets eliminated and
63 /// it's each instance is replaced by `i^th` division. If it returns `false`,
64 /// the divisions are not merged. `merge` can also do side effects, For example
65 /// it can merge the local identifiers in IntegerPolyhedron.
66 void removeDuplicateDivs(
67  std::vector<SmallVector<int64_t, 8>> &divs,
68  SmallVectorImpl<unsigned> &denoms, unsigned localOffset,
69  llvm::function_ref<bool(unsigned i, unsigned j)> merge);
