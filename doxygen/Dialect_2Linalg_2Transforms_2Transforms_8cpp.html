Helper function that tries to pad opOperand.

Exit early for scalar operands, if paddingFunc returns failure, or if opOperand is not defined by an ExtractSliceOp. Otherwise, try to pad the operand even if it already has a static shape. Set result to the result of the created tensor::PadOp or and return success if the operand either has been padded to a static shape or already had a static shape and failure otherwise.
