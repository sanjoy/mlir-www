22 /// An integer polyhedron is the set of solutions to a list of affine
23 /// constraints over n integer-valued variables/identifiers. Affine constraints
24 /// can be inequalities or equalities in the form:
25 ///
26 /// Inequality: c_0*x_0 + c_1*x_1 + .... + c_{n-1}*x_{n-1} + c_n >= 0
27 /// Equality : c_0*x_0 + c_1*x_1 + .... + c_{n-1}*x_{n-1} + c_n == 0
28 ///
29 /// where c_0, c_1, ..., c_n are integers.
30 ///
31 /// Such a set corresponds to the set of integer points lying in a convex
32 /// polyhedron. For example, consider the set: (x, y) : (1 <= x <= 7, x = 2y).
33 /// This set contains the points (2, 1), (4, 2), and (6, 3).
34 ///
35 /// The integer-valued variables are distinguished into 3 types of:
36 ///
37 /// Dimension: Ordinary variables over which the set is represented.
38 ///
39 /// Symbol: Symbol variables correspond to fixed but unknown values.
40 /// Mathematically, an integer polyhedron with symbolic variables is like a
41 /// family of integer polyhedra indexed by the symbolic variables.
42 ///
43 /// Local: Local variables correspond to existentially quantified variables. For
44 /// example, consider the set: (x) : (exists q : 1 <= x <= 7, x = 2q). An
45 /// assignment to symbolic and dimension variables is valid if there exists some
46 /// assignment to the local variable `q` satisfying these constraints. For this
47 /// example, the set is equivalent to {2, 4, 6}. Mathematically, existential
48 /// quantification can be thought of as the result of projection. In this
49 /// example, `q` is existentially quantified. This can be thought of as the
50 /// result of projecting out `q` from the previous example, i.e. we obtained {2,
51 /// 4, 6} by projecting out the second dimension from {(2, 1), (4, 2), (6, 2)}.
52 ///
