Detailed Description

template<typename OpTy>
struct mlir::bufferization::AllocationHoistingBarrierOnly< OpTy >

AllocationHoistingBarrierOnly is an external implementation of BufferizableOpInterface for ops that are (not yet) bufferizable, but are known to be allocation hoisting barriers.

All interface methods (except for isAllocationHoistingBarrier) are implemented conservatively.
